# 💰 Expense Tracker System



<p align="center">
  <b>Smart Expense Management System with Budget Tracking & PDF Reports 📊</b>
</p>

---

## ✨ Overview

A **console-based Expense Tracker System** built using **Python + SQLite**, designed to help users:

- Track daily expenses 💸  
- Manage budgets 📊  
- Get smart spending insights 🤖  
- Generate monthly PDF reports 📄  

---

## 🚀 Features

### 🔐 Authentication
- 👤 User Registration  
- 🔑 Secure Login System  
- 🛡️ User-based data isolation  

---

### 💸 Expense Management
- ➕ Add Expenses (Date, Category, Description, Amount)  
- 📋 View All Expenses  
- ✏️ Update Expenses  
- ❌ Delete Expenses  
- 🔍 Search by Category  

---

### 📊 Budget System
- 💰 Set Monthly Budget  
- 📉 Track Total Spending  
- 📊 View Remaining Budget  

---

### 🤖 Smart Budget Advisor
- 🚨 Over Budget Alerts  
- ⚠️ 90% Usage Warning  
- 🟡 70% Usage Caution  
- 💡 Smart spending suggestions  

---

### 📄 PDF Reporting
- 📑 Monthly Expense Reports  
- 📊 Category-wise Breakdown  
- 🧾 Transaction History  
- 💰 Budget Summary
